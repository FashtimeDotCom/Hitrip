本系统UI基于mui框架+bootstrap框架+Jquery.js插件开发；遵循本原则将让您的开发工作事半功倍。

各插件版本说明：

1、Bootstrap相关：V3.3.5（JS文件、CSS文件等，默认支持glyphicons-halflings-regular字体图标）
2、Fontawesome相关：V3.2.1（CSS文件、字体文件等）
3、Jquery：V2.2.3
4、Jquery UI：V1.10.2
5、jQuery UI Touch Punch：V 0.2.2
6、MUI相关：V3.3.0（JS文件、CSS文件等）
7、其他：
文档结构说明（点击展示）
  1. 基本原则：框架使用了$do \ $ac \ $operation 的路由方式；程序文件、模板文件，都大致上遵循了该方式进行相应文件的命名。
  2. core：主要核心程序文件目录
    1. app：手机端功能程序；
    2. web：后台管理及核心业务处理程序；
    3. appweb：前端核销管理功能程序；
    4. function：常用工具函数文件集合；
    5. class：常用工具类文件集合；
    6. model：常用业务模型文件集合；
    7. public：公用文件集合；
    8. msgtpls：消息模板文件存档
    9. status：状态集文件存档
    10. tables：数据表处理文件集合
  3. plugin：插件程序及插件文件；
  4. resource：资源文件夹，如样式表、默认图片包、js脚本等
    1. components：组件资源
    2. common：通用类型的资源
    3. images：通用图片包
    4. uphelpsql：后台自助升级程序调用的升级sql命令
  5. template：风格模板文件，按模板名称对应文件夹
    1. web：后台风格
      • default：默认风格
        1. common：通用的、重复使用的，如头部、脚部文件、页卡导航条、样式文件等；
    2. mobile：前台风格
      • default：默认风格；新添加风格时，可复制本文件夹重命名后直接进行具体修改，请务必保正各文件的完整性
        1. 453：模板内用的一些资源文件；
        2. common：通用的、重复使用的，如头部、脚部文件、页卡导航条、样式文件等；
    3. help：帮助文件资料包，操作指引、开发文档、状态说明等。
    4. modset：模块设置文件。
